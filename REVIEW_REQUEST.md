- 기존: 인스타그램 인증 항목과 알림 설정 항목 사이 간격이 다른 항목보다 큼
- 수정: 4개 항목(인스타그램 인증, 알림 설정, 의견 보내기, 앱 소문내기) 간격 동일하게 통일

## 검수 4 — 메시지 기능 추가 (2026-04-14)

[업데이트 내용]

1. 메시지 기능 추가
- 알람 추가 시 상대에게 메시지를 함께 보낼 수 있는 입력란 추가
- 메시지 페이지 신설: 보낸 메시지 / 받은 메시지 탭 구분
- 빈 상태 안내 문구 및 알람 추가 바로가기 버튼 제공

2. 좋아하는 사람 수 바 문구 개선

- 피드백 페이지와 홈의 바텀시트 CSS 클래스 분리로 스타일 충돌 해소
- 알림 받기 다이얼로그가 바텀 카드 형태로 정상 표시
