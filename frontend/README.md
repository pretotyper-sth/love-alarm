# Frontend

프론트엔드 애플리케이션 코드입니다.

## 구조

```
frontend/
├── src/              # 소스 코드
│   ├── pages/        # 페이지 컴포넌트
│   ├── components/   # 재사용 가능한 컴포넌트
│   ├── utils/        # 유틸리티 함수
│   └── ...
├── public/           # 정적 파일
├── index.html        # HTML 진입점
├── vite.config.js    # Vite 설정
└── eslint.config.js  # ESLint 설정
```

## 실행

프로젝트 루트에서 다음 명령어를 실행하세요:

```bash
npm run dev    # 개발 서버 실행
npm run build  # 프로덕션 빌드
```

## 기술 스택

- React 18.3.1
- Vite 7.2.4
- TDS Mobile (@toss/tds-mobile)
- React Router DOM
